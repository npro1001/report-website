Glossary

  • Term: Security Integrated and Event Management (SIEM)
    Definition: A platform that allows aggregated security logs to be collected and reviewed
  • Term: EDR/XDR
    Definition: Endpoint detection and response - advanced antivirus with central management capabilities
  • Term: Server Firewall
    Definition: A computer usually without a monitor used to host content on the internet such as websites
  • Term: Antivirus Domain/subdomain
    Definition: Network layer software or hardware that allows some network requests in and out and denies others
  • Term: IDS/IPS
    Definition: Software program that looks at compiled code that contain malicious instructions
  • Term: PII
    Definition: Personally Identifiable Information. Any information that can link back to an individual
  • Term: Ransomware
    Definition: Specific computer malware that encrypts hard drive and demands ransom to get decryption key
  • Term: POS System
    Definition: Point-of-sale system, used for checkouts in stores and food.
  • Term: Intranet
    Definition: Network that is usually private. Targeting of a user-group within an organization to attempt to get a user to perform malicious action.
  • Term: Phishing
    Definition: Cybersecurity and infrastructure security agency. Government agency responsible for ensuring cybersecurity resources to the general public.
  • Term: CIA Triad
    Definition: Confidentiality, Integrity, and Availability. Key tenant to IT security
  • Term: VPN
    Definition: Virtual Private Network, allows for IP address to appear to originate from a different location ensuring anonymity
  • Term: CISA
    Definition: Cybersecurity and infrastructure security agency. Government agency responsible for ensuring cybersecurity resources to the general public.
  • Term: Keylogger
    Definition: Device, either software or hardware, that allows complete capture of keystrokes and sometimes mouse movements.
  • Term: FIM
    Definition: File integrity monitoring, integral part of EDR that checks for important operating system files being modified
  • Term: Encryption
    Definition: Cryptographic representation of important data that can not be read by computers or humans unless decrypted with correct key
